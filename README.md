# Project Title

XR-android-driver

## Description

A project to allow usage of Xreal Air (2) (and Potentially other) Glasses on android devices.

## Getting Started

### Dependencies

- Android studio
- ahrs
- Chaquopy
- Ndk

### Installing

Clone the repository into android studio, and build it for your device.
